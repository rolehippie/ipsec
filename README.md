## Requirements

- Minimum Ansible version: `2.10`

## Default Variables

### ipsec_charon_debug

Debugging levels for charon

#### Default value

```YAML
ipsec_charon_debug: ike 1, knl 1, cfg 0
```

### ipsec_sites_extra

List of extra sites

#### Default value

```YAML
ipsec_sites_extra: []
```

### ipsec_sites_general

List of general sites

#### Default value

```YAML
ipsec_sites_general: []
```

### ipsec_unique_ids

Allow or disallow multiple ids

#### Default value

```YAML
ipsec_unique_ids: false
```
